# Thu muc dataset - Chapter 5: Face Recognition

Day la noi luu tru anh tham chieu de dang ky khuon mat.

## Cach su dung:
1. Chay cell `capture_reference_images()` trong notebook de chup anh tu webcam
2. Hoac copy anh khuon mat vao thu muc nay (dinh dang .jpg, .png)
3. Ten file (khong bao gom phan mo rong) se duoc dung lam ten nguoi

## Vi du:
- `nguyen_van_a.jpg` -> ten nguoi: `nguyen_van_a`
- `tran_thi_b.png`   -> ten nguoi: `tran_thi_b`

## Luu y:
- Moi anh chi nen chua 1 khuon mat
- Dam bao anh sang tot, khuon mat nhin thang
- FaceNet yeu cau khuon mat duoc crop va resize ve 160x160 pixels
