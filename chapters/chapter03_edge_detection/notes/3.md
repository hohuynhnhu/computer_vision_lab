# Bài toán 2 - Canny + Nhận dạng hình dạng

## Mục tiêu

Phát hiện:

* Hình tròn
* Hình chữ nhật
* Đa giác

---

## Quy trình

### Bước 1

Phát hiện cạnh bằng Canny.

### Bước 2

Tìm contour:

```python
cv2.findContours()
```

### Bước 3

Xấp xỉ contour:

```python
cv2.approxPolyDP()
```

### Bước 4

Phân loại hình dạng:

* 3 đỉnh → Tam giác
* 4 đỉnh → Hình chữ nhật / hình vuông
* > 4 đỉnh → Đa giác
* Hình tròn → Circularity

### Bước 5

Vẽ nhãn lên ảnh.

---

## Kết quả mong đợi

Hiển thị:

* Ảnh gốc
* Ảnh cạnh Canny
* Contour tìm được
* Hình dạng được nhận diện
